# Sky: A 位-Calculus Interpreter

Little project in implementing some 位-calculus bits. What I'm planning to 
eventually do is to make:

1. [ ] An untyped 位-calculus interpreter.
2. [ ] A lisp->位 compiler with recursion for funsies.
3. [ ] A simply-typed 位-calc interpreter.

I'm intending to follow the formal definitions as closely as possible. Maybe 
I'll rewrite this in Coq someday so I can prove some things about it formally
